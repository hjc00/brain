# Unity UI 开发提效新方案：蓝湖设计稿一键生成 Prefab 的完整工作流实战

## 概述

本文分享了一种基于 **VectoUI** 的自动化 UI 工作流，可将蓝湖设计稿一键转化为 Unity Prefab，将原本以天为单位的工作量压缩到秒级，效率提升约 **80%**。

---

## 传统工作流的痛点

| 痛点 | 说明 |
|------|------|
| 切图管理繁琐 | 手动下载、解压、导入、设置 Texture Type 和九宫格 |
| 坐标系差异 | 设计软件与 Unity UGUI 坐标系不同，需人肉计算偏移 |
| 体验割裂 | 开发者被迫沦为"UI 纺织工"，精力被拼图消耗 |

---

## 解决方案：VectoUI 跨平台 UI 编译引擎

VectoUI 是一个跨平台 UI 编译引擎，在云端将设计稿转化为中间标准化格式，再编译生成目标引擎的原生预制体。

**核心特点：**
- 设计师零学习成本，无需改变现有习惯
- 开发者一键生成，所见即所造

---

## 完整工作流

### Step 1: 设计师操作

1. 在 PS/Sketch 完成界面设计
2. 标记切图（按钮、底板、图标等）
3. 通过蓝湖插件上传到蓝湖云端

### Step 2: 开发者配置

1. 前往 [vectoui.com](https://vectoui.com/) 注册并获取 Unity Beta 插件
2. 导入 Unity 工程，打开 VectoUI 控制面板
3. 输入蓝湖凭证，同步设计稿列表

### Step 3: 一键生成

点击 **"生成 UI"** 按钮，5-10 秒内自动完成：

- **资源自动化**：自动拉取高清切图，转化为 Sprite
- **节点树构建**：生成与设计稿层级一致的 UGUI 节点树
- **100% 视觉还原**：精准还原 Position、Size、Pivot、Anchors

---

## 进阶特性

### 1. TextMeshPro 深度支持

- 自动映射预配置的 TMP 字体资产
- 精准还原字号、颜色、行高、对齐

### 2. 自定义后处理规则 (Post-processing)

基于命名规范的自动化脚本绑定：

| 规则 | 效果 |
|------|------|
| `Btn_` 前缀 | 自动挂载 Button 组件 + 缩放动效 |
| `ScrollView_` 前缀 | 自动生成 ScrollRect + Mask |
